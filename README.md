# AML Risk Intelligence Engine

An Explainable AI-based Mule Account Detection System with SHAP-driven risk interpretation.

## Features

- Behavioral Risk Scoring
- SHAP Explainability
- AML Compliance Scorecard
- Risk Intelligence Dashboard
- Synthetic Data Training Pipeline
- XGBoost Model

## Tech Stack

Frontend:
- React
- TypeScript
- TailwindCSS
- Axios

Backend:
- FastAPI
- XGBoost
- SHAP
- Scikit-Learn

## How to Run

in vs code terminal -> git clone url of this repo

### Backend
cd backend
uvicorn app:app --reload

### Frontend
cd frontend
npm install
npm run dev
